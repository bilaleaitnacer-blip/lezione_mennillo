# settare ambiente linux per git 
![Git Icona](./img/GitHub_logo_2013.webp)
instalare git sulla machina con `sudo apt instal git`
settare username e password di github con i seguenti comandi 
`git config --global user.name "tuousernamegithub"`
`git config --global user.email "tuauseremailgithub"`
